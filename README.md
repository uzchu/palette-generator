# palette-generator
Using deep learning method(GAN) to generate color palette


## Compatibility

Python 3.6  
keras 2.2.0  
tensorflow 1.14.0



## Update

| Date       | Update       |
| ---------- | ------------ |
| 2019-08-28 | First upload |

## Data

자체 데이터를 교육하려면 palettes 폴더의 형식을 참조하십시오.



## Train

작업은 jupyter 파일의 단계를 참조하십시오.

## Test

```
python color_generator.py your_weight_path
```
